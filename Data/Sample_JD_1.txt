About the Role

Perform data cleaning, preprocessing, and feature engineering
Build predictive models using Scikit-learn, TensorFlow, or PyTorch
Analyze large datasets using Python, Pandas, NumPy
Write complex queries using SQL
Create dashboards and reports using Power BI or Tableau
Collaborate with backend teams to deploy models using FastAPI or Flask
Work with cloud platforms like AWS
Use Docker for containerization
Apply statistical techniques and hypothesis testing

Required Skills -
Strong proficiency in Python
Experience with Machine Learning & Deep Learning
Hands-on experience with Scikit-learn
Good knowledge of SQL
Experience with TensorFlow or PyTorch
Knowledge of Data Visualization tools
